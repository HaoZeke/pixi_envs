* Step 1: Prepare Input Structures

** Understanding .con Files

eOn uses =.con= format for atomic structures:

#+begin_src text
# Comment line
3  # Number of atoms
H   0.000000   0.000000   0.000000
C   0.000000   0.000000   1.065000
N   0.000000   0.000000   2.220000
#+end_src

** Example: HCN Isomerization

We'll use the HCN → HNC isomerization example:

#+begin_src bash
# Copy example structures
cp -r examples/hcn_isom ~/my_neb_project/
cd ~/my_neb_project
#+end_src

** Verify Structures

Check your structures with ASE or visualization software:

#+begin_src python
from ase.io import read
hcn = read('hcn.con')
hnc = read('hnc.con')
print(f"HCN atoms: {len(hcn)}")
print(f"HNC atoms: {len(hnc)}")
#+end_src

* Step 2: Configure the Workflow

** Copy Configuration Template

#+begin_src bash
cp examples/hcn_isom/config.yaml my_config.yaml
#+end_src

** Edit Configuration

Open =my_config.yaml= and update paths:

#+begin_src yaml
systems:
  hcn_isom:
    reactant: /home/yourname/my_neb_project/hcn.con
    product: /home/yourname/my_neb_project/hnc.con
    use_ira: true

compute:
  device: cuda  # or 'cpu' if no GPU
#+end_src

** Key Parameters to Review

| Parameter | Default | When to Change |
|-----------+---------+----------------|
| =images= | 18 | Increase for complex reactions |
| =converged_force= | 0.0514221 | Reduce for higher precision |
| =max_iterations= | 1000 | Increase if not converging |
| =device= | cuda | Set to =cpu= if no GPU |

** Interpret Energy Profile

Key features to identify:
1. *Reactant energy*: Reference point (0.0 eV)
2. *Product energy*: Relative stability (~0.6 eV for HNC)
3. *Maximum energy*: Saddle point (~1.8 eV)
4. *Barrier height*: E_saddle - E_reactant

** View Plots

#+begin_src bash

# Open 2D landscape

* Step 5: Validate Results

** Check Convergence

#+begin_src bash
# Check final forces
tail -20 results/neb/hcn_isom/neb.dat
#+end_src

Look for:
- RMS force < 0.0514221 eV/Å (convergence criterion)
- Maximum force < 0.1 eV/Å
- Energy change < 0.001 eV between iterations

** Compare with Literature

For HCN isomerization:
| Method | Barrier (eV) |
|--------+--------------|
| PET-MAD | ~1.8 |
| CCSD(T) | 1.79 |
| DFT/B3LYP | 1.85 |

Your result should be within ±0.2 eV of these values.

* Troubleshooting

** Problem: IRA Alignment Fails

*Symptoms*: Error in =align_endpoints= rule

*Solutions*:
1. Increase =alignment.kmax= to 3.0
2. Check atom ordering is consistent
3. Verify structures are reasonable (no overlapping atoms)

** Problem: NEB Does Not Converge

*Symptoms*: Maximum iterations reached, forces still high

*Solutions*:
1. Increase =max_iterations= to 2000
2. Reduce =max_move= to 0.05
3. Add more images (24 instead of 18)
4. Check endpoints are properly minimized

** Problem: Out of Memory (GPU)

*Symptoms*: CUDA out of memory error

*Solutions*:
1. Switch to CPU: =device: cpu=
2. Reduce number of images
3. Use smaller model (if available)
